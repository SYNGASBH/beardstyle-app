# Soul Patch – Inženjerski profil i karakteristike

Soul Patch je minimalistička facial hair opcija koja se sastoji od malog područja dlaka direktno ispod donje usne. Poznat i kao "mouche", "jazz dot", ili "flavor saver", ovaj stil je popularan od 1950-ih jazz ere i ostao je simbol cool, laid-back estetike. Soul Patch je najmanji beard stil koji se još uvijek kvalificira kao "facial hair".

---

## Ključne dimenzije i proporcije

* **Širina**: 1–2.5 cm
  → Narrow: 1–1.5 cm
  → Standard: 1.5–2 cm
  → Wide: 2–2.5 cm
  → Ne šire od 2.5 cm

* **Visina**: 1–3 cm
  → Short: 1–1.5 cm
  → Medium: 1.5–2 cm
  → Extended: 2–3 cm
  → Može se spojiti sa chin hair

* **Pozicija**: Direktno ispod centra donje usne
  → Centered on mentolabijalni sulkus
  → Gap od lip: 2–5 mm

* **Oblik**: Variable
  → Round/circular
  → Triangular (pointing down)
  → Square/rectangular
  → Natural

* **Gustina**: Solid
  → Fill factor: >90% za defined patch
  → Patchy soul patch izgleda accidental

* **Ostalo lice**: Clean
  → No mustache
  → No chin beard
  → No stubble
  → Complete isolation

---

## Održavanje – Kontrolne tačke

* **Trim interval**:
  * Soul patch shape: Svakih 3–5 dana
  * Face shave: Svakih 1–2 dana
  * **Ukupno**: Srednje održavanje (3–4 sessiona sedmično)

* **Alati**:
  * Precision trimmer (0 mm) – za edges
  * Detail trimmer
  * Razor – za face
  * Small scissors – za length

* **Proizvodi**:
  * Minimal products needed
  * Aftershave: Za shaved areas
  * Conditioning balm: Optional

* **Vrijeme po sessiji**: 5–8 minuta
* **Sedmični time investment**: 20–30 minuta

## Razlike vs slični stilovi

| Karakteristika | Soul Patch | Goatee | Van Dyke | Circle Beard | Stubble |
|----------------|------------|--------|----------|--------------|---------|
| **Size** | Tiny patch | Chin coverage | Chin + mustache | Chin + connected mustache | Full face |
| **Coverage** | Below lip only | Chin | Chin + lip | Chin + lip | Full |
| **Mustache** | No | Optional | Yes (separated) | Yes (connected) | Integrated |
| **Maintenance** | Medium | Medium | High | Medium | Medium |
| **Vibe** | Jazz cool | Minimalist | Artistic | Casual | Rugged |

### Ključne distinkcije:
* **Soul Patch vs Goatee**: Soul Patch je samo ispod usne; Goatee pokriva chin.
* **Soul Patch vs Stubble**: Soul Patch je isolated spot; Stubble je full face coverage.
* **Soul Patch vs Flavor Saver**: Same thing – different colloquial names.

---

## Dizajn pravila za autentičan Soul Patch

1. **Size must be small** – Larger than 2.5 cm širine prelazi u extended goatee territory.

2. **Centered perfectly** – Off-center soul patch izgleda kao mistake.

3. **Clean isolation** – Rest of face must be clean. Any other facial hair changes the style.

4. **Shape consistency** – Pick a shape, maintain it consistently.

5. **Dense fill required** – Sparse soul patch looks accidental, not intentional.

6. **Clean edges** – Defined boundaries essential.

7. **Regular shaving around** – Daily ili every-other-day face maintenance.

8. **Subtle is the goal** – Soul patch je accent, ne statement.

## Growth Timeline

**Od clean shaven do Soul Patch:**

### Phase 1: Initial Growth (Sedmica 1–2)
* **Sedmica 1**:
  → Expected growth: ~1 cm
  → Define patch zone
  → Shave rest of face
* **Sedmica 2**:
  → Expected growth: ~1.5–2 cm
  → Soul patch established
  → Shape edges

### Phase 2: Maintenance (Sedmica 3+)
* Steady state quickly achieved
* Regular face shaving
* Patch maintenance svakih 3–5 dana

**Total time to achieve**: 1–2 sedmice
**Minimum growth required**: 0.5–1 cm

## Transition Paths

### 1. Clean Shaven → Soul Patch
* **Duration**: 1–2 sedmice
* **Difficulty**: Easy
* **Key steps**:
  1. Define patch zone
  2. Shave rest of face
  3. Let patch grow
  4. Shape when visible

### 2. Stubble → Soul Patch
* **Duration**: 1 session
* **Difficulty**: Easy
* **Steps**:
  1. Shave everything except patch zone
  2. Define patch shape
  3. Immediate soul patch

### 3. Goatee → Soul Patch
* **Duration**: 1 session
* **Difficulty**: Easy
* **Steps**:
  1. Shave chin
  2. Keep only patch zone
  3. Shape to soul patch dimensions

### 4. Soul Patch → Goatee
* **Duration**: 3–4 sedmice
* **Difficulty**: Easy
* **Steps**:
  1. Let chin grow
  2. Soul patch expands into goatee
  3. Shape when 2+ cm

### 5. Soul Patch → Van Dyke
* **Duration**: 5–6 sedmice
* **Difficulty**: Medium
* **Steps**:
  1. Grow chin
  2. Grow mustache separately
  3. Style into Van Dyke
